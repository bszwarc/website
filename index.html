A flexible and easy way to connect and extend enterprise applications in a cloud-native world

Kyma mission: Make simple things simple again!

Kyma aims to equip enterprise software users with agile tools to customize, extend, and integrate any API-enabled application and tailor it to their needs.

Kyma makes the development and operation of microservices-based solutions easy by providing consistent service management capabilities and the integration of the existing enterprise software in a service-oriented manner.

What is Kyma?

Kyma is a cloud-native application development framework.

It provides the last mile capabilities that a developer needs to build a cloud-native application using several open-source projects such as Kubernetes, Istio, NATS, Kubeless, and Prometheus, to name a few.

It is designed natively on Kubernetes and, therefore, it is portable to all major cloud providers.
